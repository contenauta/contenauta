# Contenauta

**La tua bussola editoriale.**

Contenauta distilla anni di esperienza in strategia editoriale in un file HTML gratuito. Rispondi a qualche domanda sulla tua attività e in pochi secondi ottieni pilastri editoriali, matrice dei contenuti e calendario mensile — pronti da usare, esportabili in CSV.

Creato da [Giorgia Martinucci](https://www.giorgiamartinucci.com/it/), consulente in comunicazione per web agency e imprese dal 2009.

---

## Come funziona

Contenauta è un file HTML che funziona direttamente dentro **Claude.ai**, l'AI di Anthropic. Non serve installare nulla, non serve creare account extra — se hai già Claude.ai, sei pronto.

Il file guida l'utente attraverso un form in 4 step. Una volta completato, Claude genera:

- **Scheda attività** — un ritratto sintetico della tua presenza online
- **Pilastri editoriali** — i temi portanti della tua comunicazione
- **Matrice dei contenuti** — idee organizzate per pilastro e obiettivo
- **Calendario mensile** — 20 contenuti pronti, distribuiti su 4 settimane
- **Sviluppa un post** — genera il testo di un post partendo dalla matrice, per ogni piattaforma e formato

Puoi generare il calendario per i mesi successivi senza ricominciare da capo, e tutto è esportabile in CSV.

---

## Come si usa

### 1. Scarica il file

Scarica il file `contenauta.html` dalla [pagina Release](https://github.com/contenauta/contenauta/releases/download/v1.0/contenauta.html).

### 2. Apri Claude.ai

Vai su [claude.ai](https://claude.ai) e accedi con il tuo account. Se non ce l'hai, puoi crearne uno gratuitamente.

### 3. Carica il file nella chat

Apri una nuova conversazione e allega il file HTML appena scaricato usando il pulsante di allegato (📎). Poi scrivi qualcosa come:

> *"Mostra come artifact"*

oppure semplicemente invia il file — Claude lo riconoscerà e lo aprirà come strumento interattivo.

### 4. Compila il form e genera la strategia

Segui i 4 step del wizard, poi clicca **Genera la strategia**. In circa 30 secondi avrai tutto pronto.

---

## Cosa puoi caricare nel form

Nel primo step puoi allegare un file con materiali sulla tua attività — **PDF o immagini** (JPG, PNG, WebP). Brochure, listino, presentazione: tutto quello che aiuta Claude a capire meglio di cosa ti occupi.

*Nota: i file .docx o .txt non sono supportati. Se hai un documento Word, convertilo in PDF prima di caricarlo.*

## Funzionalità principali

| Funzionalità | Descrizione |
|---|---|
| Wizard in 4 step | Form guidato pensato anche per chi non ha esperienza di marketing |
| Pilastri editoriali | Da 3 a 6 temi portanti personalizzati |
| Matrice dei contenuti | Idee per ogni pilastro × obiettivo (posizionamento, autorevolezza, educazione, conversione) |
| Calendario mensile | 4 settimane, lun-ven, 20 contenuti |
| Mesi successivi | Genera i mesi 2, 3, 4... con idee sempre nuove |
| Sviluppa un post | Testo pronto per Instagram, LinkedIn, Facebook, TikTok, Twitter/X — con suggerimenti sul formato visual |
| Export CSV | Scarica matrice e calendario in CSV |

---

## Note

Contenauta usa le API di Claude tramite il proxy di Claude.ai. Questo significa che **non ci sono costi aggiuntivi** per chi lo usa: ogni utente utilizza il proprio account Claude, e la generazione rientra nell'utilizzo normale della piattaforma.

Il file non raccoglie dati, non si connette a server esterni e non richiede registrazioni.
